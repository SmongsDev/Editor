# Visual Text Editor 설명서

## Window OS

### 기본 환경 세팅
+ gcc install
+ make install
+ curses.h, pdcurses.a, pdcurses.dll 필요

### 실행 파일 생성
명령어 : make

### 실행 파일 및 오브젝트 파일 삭제
명령어 : make clean

## Linux & MAC OS

### 기본 환경 세팅
+ gcc
+ make
+ ncurses.h

### 실행 파일 생성
명령어 : make

### 실행 파일 및 오브젝트 파일 삭제
명령어 : make clean
